MaklaPlace is a modular WordPress marketplace plugin for independent chefs.

== Description ==
This release is the plugin foundation only.

== Installation ==
1. Upload the plugin folder to `wp-content/plugins/`.
2. Activate the plugin from the WordPress admin.
